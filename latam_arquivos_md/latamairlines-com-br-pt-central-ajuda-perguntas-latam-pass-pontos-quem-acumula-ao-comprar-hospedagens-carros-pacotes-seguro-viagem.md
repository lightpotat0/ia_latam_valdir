Ao reservar ou comprar um complemento de viagem para mais de uma pessoa, quem acumula as Milhas LATAM Pass e soma Pontos Qualificáveis?
=======================================================================================================================================

Os complementos de viagem são serviços adicionais à sua passagem, como hospedagem, aluguel de carros ou seguro viagem.

O crédito das milhas dependerá do parceiro com quem a reserva foi feita. Confira os detalhes na tabela abaixo:

|  |  |  |
| --- | --- | --- |
| **Parceiro** | **Complemento no Destino** | **Quem Acumula** |
| Booking | Hotel | Hóspede principal |
| Despegar | Voo | Cada passageiro que voar |
|  | Hotel | Titular da reserva |
| Pacotes de Viagem LATAM (Venda com assistência de um assessor) | Voo | Cada passageiro que voar |
|  | Hotel | Hóspede principal |
| Assist Card | Seguro viagem | Beneficiário do número LATAM Pass informado |
| Rentalcars | Aluguel de carros | Condutor principal |
| Mozio | Traslado | Beneficiário do número LATAM Pass informado |
| Aluguel de carros LATAM | Aluguel de carros | Condutor principal |

**Importante:** Insira o número do cliente LATAM Pass ao realizar a compra para garantir o crédito de Milhas LATAM Pass e Pontos Qualificáveis.
