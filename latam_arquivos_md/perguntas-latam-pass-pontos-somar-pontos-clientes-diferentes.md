Posso resgatar uma passagem somando Milhas LATAM Pass de dois clientes diferentes?
==================================================================================

Não, só é possível resgatar passagens com as milhas de um cliente. No entanto, você pode [transferir milhas](https://www.pontosmultiplus.com.br/latampass/acelere-seus-pontos/transferir-pontos) para a conta de outro cliente para que este tenha as milhas suficientes para fazer o resgate.
