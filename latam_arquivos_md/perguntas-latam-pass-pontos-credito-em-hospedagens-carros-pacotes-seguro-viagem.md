Quanto tempo demora para as Milhas LATAM Pass e Pontos Qualificáveis serem creditados ao comprar complementos de viagem?
========================================================================================================================

Os complementos de viagem são serviços adicionais à sua passagem que você compra ou reserva, como hospedagem, aluguel de carros ou seguro viagem.

Os prazos de crédito de Milhas LATAM Pass e Pontos Qualificáveis variam conforme o parceiro. Confira os detalhes na tabela a seguir:

|  |  |  |
| --- | --- | --- |
| **Parcerias** | **País** | **Prazo para Crédito** |
| Booking.com, Rentalcars | Chile, Peru, Argentina, Colômbia, Equador, Paraguai, Uruguai, México | 64 dias após o checkout |
| Despegar | Chile, Peru, Argentina, Colômbia, Equador, México | 45 dias após o checkout |
| Pacotes de Viagem LATAM (Venda com assistência de um assessor) | Chile, Peru, Paraguai | 45 dias após o checkout |
| Assist Card | Chile, Peru, Colômbia, Equador, Paraguai, Argentina | 45 dias após o checkout |
| Mozio Aluguel de carro LATAM | Chile, Peru, Colômbia, Equador | 45 dias após o checkout |
